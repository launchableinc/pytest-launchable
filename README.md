# Usage

For usage instructions, see [https://www.launchableinc.com/docs/resources/integrations/pytest/](https://www.launchableinc.com/docs/resources/integrations/pytest/)

# Development
We recommend Pipenv
```shell
pip install pipenv
pipenv install --dev
```

## Load development environment
```shell
pipenv shell
```

## Run tests
```shell
pipenv run test
```

## Run lint and format
```shell
# lint
pipenv run lint

# format
pipenv run format
```
